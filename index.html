Política de Privacidade e Uso de Dados

Última atualização: 06 de maio de 2026

Esta Política de Privacidade aplica-se especificamente ao serviço de agendamento de visitas e validação de identidade. Para informações completas sobre o tratamento de dados pessoais pela AGSinn, consulte a Política de Privacidade institucional disponível abaixo.

A Política de Privacidade completa da AGSinn pode ser acessada em: Política de Privacidade Institucional

1. Introdução

Controlador: AGS Soluções de Automação e Segurança LTDA (AGSinn)

CNPJ: 18.828.193/0001-39

Endereço: Belo Horizonte, Minas Gerais

A AGSinn realiza o tratamento de dados pessoais com o objetivo de viabilizar o agendamento de visitas e a validação de identidade dos titulares dos dados, em conformidade com a Lei nº 13.709/2018 (Lei Geral de Proteção de Dados - LGPD).

2. Dados Pessoais Tratados

Para execução dos serviços, são tratados, quando necessário, os seguintes dados pessoais:

  • CPF
  • Nome completo
  • Data de nascimento
  • Foto
  • Dados constantes na CNH

3. Finalidade do Tratamento

Os dados são utilizados exclusivamente para:

  • Validação da identidade do titular de dados pessoais por meio da API Datavalid (SERPRO/SENATRAN)
  • Agendamento de visitas
  • Prevenção a fraudes e garantia da segurança do titular

4. Base Legal

O tratamento de dados pessoais é realizado com base nas seguintes hipóteses legais:

  • Consentimento do titular de dados pessoais, quando aplicável
  • Execução de procedimentos preliminares relacionados ao serviço solicitado
  • Atendimento a obrigações legais ou regulatórias, quando aplicável
  • Legítimo interesse (prevenção à fraude e segurança)

5. Compartilhamento de Dados

Os dados poderão ser compartilhados com o Serviço Federal de Processamento de Dados (SERPRO), que atua como operador de dados, mediante contrato, realizando o tratamento em nome da AGSinn - conforme suas instruções e em conformidade com a legislação aplicável -, para consulta junto às bases da Secretaria Nacional de Trânsito (SENATRAN), exclusivamente para validação de identidade.

6. Armazenamento e Segurança

A AGSinn adota medidas técnicas e administrativas adequadas, incluindo controles de acesso, monitoramento e, quando aplicável, técnicas de criptografia para proteger os dados pessoais contra acessos não autorizados, perda, alteração ou divulgação indevida.

7. Retenção de Dados

Os dados pessoais serão armazenados pelo tempo necessário ou enquanto durar a relação com o titular, para cumprir as finalidades descritas nesta política, respeitando os prazos legais e regulatórios aplicáveis, ou até a revogação do consentimento pelo titular de dados pessoais, quando aplicável.

8. Direitos do Titular

O titular de dados pessoais pode, a qualquer momento, solicitar:

  • Confirmação da existência de tratamento
  • Acesso aos dados
  • Correção de dados incompletos ou desatualizados
  • Exclusão de dados, quando aplicável
  • Portabilidade dos dados, quando aplicável
  • Informação sobre compartilhamento
  • Informação sobre consentimento
  • Revisão de decisões automatizadas
  • Revogação do consentimento

9. Revogação do Consentimento

A revogação do consentimento poderá impedir a continuidade do uso dos serviços que dependam da validação de identidade, incluindo o agendamento de visitas.

10. Contato

Para exercer seus direitos ou esclarecer dúvidas, entre em contato com nosso Encarregado de Proteção de Dados por meio do e-mail: protecaodedados@agsinn.com

11. Disposições Finais

Este documento deve ser interpretado em conjunto com a Política de Privacidade institucional da AGSinn. Em caso de conflito entre este documento e a política institucional, prevalecerá a política institucional.

Esta política poderá ser atualizada a qualquer momento. Caso ocorram atualizações, os usuários serão notificados por meio dos canais oficiais da empresa. Recomenda-se a revisão periódica deste documento.

Termos de Uso

Última atualização: 06 de maio de 2026

1. Objeto

Estes Termos de Uso regulam a utilização dos serviços oferecidos pela AGSinn, incluindo o agendamento de visitas e a validação de identidade dos titulares de dados pessoais.

2. Uso do Serviço

O titular de dados pessoais declara que as informações fornecidas são verdadeiras, completas e atualizadas, estando ciente de que seus dados poderão ser validados por meio de sistemas oficiais.

3. Responsabilidades

  • O titular de dados pessoais é responsável pelas informações fornecidas.
  • A empresa compromete-se a adotar medidas adequadas para proteção dos dados pessoais.
  • O titular de dados pessoais compromete-se a não utilizar o serviço para fins ilícitos, fraudulentos ou em desacordo com a legislação vigente.

4. Tratamento de Dados

O tratamento de dados pessoais será realizado conforme descrito na Política de Privacidade acima.

5. Limitações

A AGSinn não se responsabiliza por falhas decorrentes de indisponibilidade de serviços de terceiros, incluindo sistemas governamentais.

6. Disposições Gerais

O uso dos serviços implica na concordância com estes Termos e com a Política de Privacidade. Ao utilizar o serviço, o titular de dados pessoais declara que leu, compreendeu e concorda com estes Termos de Uso e com a Política de Privacidade, incluindo a autorização para validação de identidade por meio de integração com a API Datavalid (SERPRO/SENATRAN).

7. Disposições Finais

Estes termos poderão ser atualizados a qualquer momento. Caso ocorram atualizações, os usuários serão notificados por meio dos canais oficiais da empresa. Recomenda-se a revisão periódica deste documento.
