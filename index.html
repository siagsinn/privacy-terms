Política de Privacidade e Uso de Dados

1. Introdução

A AGSinn (AGS Soluções de Automação e Segurança LTDA) realiza o tratamento de dados pessoais com o objetivo de viabilizar o agendamento de visitas e a validação de identidade dos usuários, em conformidade com a Lei nº 13.709/2018 (Lei Geral de Proteção de Dados - LGPD).

2. Dados Pessoais Tratados

Para execução dos serviços, são tratados, quando necessário, os seguintes dados pessoais:

  • CPF
  • Nome completo
  • Data de nascimento
  • Foto
  • Categoria da CNH

3. Finalidade do Tratamento

Os dados são utilizados exclusivamente para:

  • Validação da identidade do usuário por meio da API Datavalid (SERPRO/SENATRAN)
  • Agendamento de visitas

4. Base Legal

O tratamento de dados pessoais é realizado com base nas seguintes hipóteses legais:

  • Consentimento do titular
  • Execução de procedimentos preliminares relacionados ao serviço solicitado

5. Compartilhamento de Dados

Os dados poderão ser compartilhados com o Serviço Federal de Processamento de Dados (SERPRO), que atua como operador no tratamento de dados, para consulta junto às bases da Secretaria Nacional de Trânsito (SENATRAN), exclusivamente para validação de identidade.

6. Armazenamento e Segurança

A AGSinn (AGS Soluções de Automação e Segurança LTDA) adota medidas técnicas e administrativas adequadas para proteger os dados pessoais contra acessos não autorizados, perda, alteração ou divulgação indevida.

7. Retenção de Dados

Os dados pessoais serão armazenados pelo tempo necessário para cumprir as finalidades descritas nesta política, respeitando os prazos legais e regulatórios aplicáveis, ou até a revogação do consentimento pelo titular, quando aplicável.

8. Direitos do Titular

O titular dos dados pode, a qualquer momento, solicitar:

  • Confirmação da existência de tratamento
  • Acesso aos dados
  • Correção de dados incompletos ou desatualizados
  • Exclusão de dados, quando aplicável
  • Portabilidade dos dados, quando aplicável
  • Revogação do consentimento

9. Revogação do Consentimento

A revogação do consentimento poderá impedir a continuidade do uso dos serviços que dependam da validação de identidade, incluindo o agendamento de visitas.

10. Contato

Para exercer seus direitos ou esclarecer dúvidas, entre em contato pelo e-mail: si.agsinn@gmail.com

Estes Termos de Uso regulam a utilização dos serviços oferecidos pela AGSinn (AGS Soluções de Automação e Segurança LTDA), incluindo o agendamento de visitas e a validação de identidade dos usuários.
